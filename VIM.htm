set 设置vim的各种参数
如果要让文件配置生效，必须使用在用户家目录创建 .vimrc 文件 ######### set nu set hls syntax on #自动语法高亮 colorscheme torte #设置配色方案 ######### :set bg=dark :set bg=light set termencoding=utf-8 set ruler 打开状态栏标尺 set history=100 设置保存命令的条数 set fenc=utf-8 设定默认解码 set confirm 设置未保存弹出提示 set hls 设置高亮查找功能 set nu 设置显示行号 :set 或 :se 会显示所有经过修改的部份，就是和默认值不一样的部份。 :set all 显示目前所有设定值内容。 set cursorline 突出显示当前行 查找
:/ 正向查找，表示查找下一个匹配 :? 反向查找，表示查找上一个匹配 替换
：s/p1/p2/g：将当前行中所有p1均用p2替代 ：n1,n2s/p1/p2/g：将第n1至n2行中所有p1均用p2替代 ：g/p1/s//p2/g：将文件中所有p1均用p2替换 vim中执行shell命令
：！ls 使用！可以调用外部shell命令 ：ls 不使用！调用的是vim的内部命令 ：！ruby% %代表当前文件名，意思是使用ruby执行本文件 ：shell 使用此命令会开始切换到shell界面，如果要返回到vim需要使用：exit ：read ！{cmd} 把命令行的标准输出定向到缓冲区 ：write ！{cmd} 把缓冲区的内容作为命令行的参数 :. .表示重复执行上次的命令 保存文本和退出
:w 保存文件但不退出 :w file 将修改保存在 file 中但不退出 :wq 或 ZZ 或 :x 保存文件并退出 :q 退出 :q! 不保存文件，退出 :e! 放弃所有修改，从上次保存文件开始再编辑 :w [filename] 把编辑的文档另存为 :r [filename] 读入另一个文档到当前 :n1,n2 w [filename] 将n1--n2行另存为 0 移动到行首 $或end 移动到行尾
