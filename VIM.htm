set 设置vim的各种参数
set history=100 设置保存命令的条数 set fenc=utf-8 设定默认解码 set confirm 设置为保存弹出提示 set hls 设置高亮查找功能 查找
:/ 正向查找，表示查找下一个匹配 :? 反向查找，表示查找上一个匹配 替换
：s/p1/p2/g：将当前行中所有p1均用p2替代 ：n1,n2s/p1/p2/g：将第n1至n2行中所有p1均用p2替代 ：g/p1/s//p2/g：将文件中所有p1均用p2替换 vim中执行shell命令
：！ls 使用！可以调用外部shell命令 ：ls 不使用！调用的是vim的内部命令 ：！ruby% %代表当前文件名，意思是使用ruby执行本文件 ：shell 使用此命令会开始切换到shell界面，如果要返回到vim需要使用：exit ：read ！{cmd} 把命令行的标准输出定向到缓冲区 ：write ！{cmd} 把缓冲区的内容作为命令行的参数 :. .表示重复执行上次的命令
